Workout API

Följande endpoints är definierade:

Endpoint Metod Beskrivning Inparametrar Svar
/activity/ GET Hämtar alla aktiviteter Status: 200
  • activities [
    • id (int)
    • name (text)
    • has_time (boolean)
    • has_distance (boolean)
    • distance_unit (text, "m"/"km")
    ]
/activity/{id} GET Hämtar en enskild aktivitet GET-data:
  • id (int)
 Status: 200
  • id (int)
  • name (text)
  • has_time (boolean)
  • has_distance (boolean)
  • distance_unit (text)
Status: 400
  • error (text)
/activity/ POST Sparar en ny aktivitet POST-data:
  • name (text)
  • has_time (boolean)
  • has_distance (boolean)
  • distance_unit (text, "m"/"km")
 Status: 201
  • id (int)
  • message (text)
Status: 400
  • error (text)
/activity/{id} PUT Redigerar en befintlig aktivitet PUT-data:
  • id (int)
  • name (text)
  • has_time (boolean)
  • has_distance (boolean)
  • distance_unit (text)
 Status: 200
  • result (boolean)
  • message (text)
Status: 400
  • error (text)
/activity/{id} DELETE Raderar en aktivitet DELETE-data:
  • id (int)
 Status: 204 Status: 400
  • error (text)
/workout/ GET Hämtar alla workouts Status: 200
  • workouts [
    • id (int)
    • activity_id (int)
    • date (date)
    • time (duration)
    • distance (decimal)
    • description (text)
    • rpe (int)
    ]
/workout/{id} GET Hämtar en enskild workout GET-data:
  • id (int)
 Status: 200
  • id (int)
  • activity_id (int)
  • date (date)
  • time (duration)
  • distance (decimal)
  • description (text)
  • rpe (int)
Status: 400
  • error (text)
/workout/ POST Sparar en ny workout POST-data:
  • activity_id (int)
  • date (date)
  • time (duration)
  • distance (decimal)
  • description (text)
  • rpe (int)
 Status: 201
  • id (int)
  • message (text)
Status: 400
  • error (text)
/workout/{id} PUT Redigerar en befintlig workout PUT-data:
  • id (int)
  • activity_id (int)
  • date (date)
  • time (duration)
  • distance (decimal)
  • description (text)
  • rpe (int)
 Status: 200
  • result (boolean)
  • message (text)
Status: 400
  • error (text)
/workout/{id} DELETE Raderar en workout DELETE-data:
  • id (int)
 Status: 204 Status: 400
  • error (text)

Saknas rätt inparametrar eller om parametrarna är av fel datatyp returneras ett JSON-objekt med en felkod (400 Bad request).
